About this project

This project was created to explore the possibilities of using AI with Claude to answer questions about space and to demonstrate my skills and passion for web development.
I was curious about using the vision aspect of Claude and AI in general in creating interactive experiences for users. It's been an interesting experiment, and I've learned a lot about web development and AI along the way.
If you want to see more of my projects or want to drop me a line, check out my website.

Technologies Used

This is a TypeScript web application that combines NASA's Astronomy Picture of the Day API with Claude's vision capabilities.

  • Runtime: Bun (server + bundler)
  • Language: TypeScript (both server and client)
  • AI: Anthropic SDK (claude-haiku-4-5-20251001)
  • Styling: Tailwind CSS (dark theme with orange accents)
  • APIs: NASA APOD API
Route Method Purpose
/ GET Serves index.html
/about GET Serves about.html
/api/apod GET Fetches random NASA APOD image
/api/ask POST Sends image + question to Claude, streams response
/client.js GET Transpiles & serves client.ts
/styles.css GET Serves compiled Tailwind CSS
